Welcome to Caravelo's market survey API!

1. Surveys

1.1. Endpoints

Our endpoints allow you to make calls to our API by various parameters, so you can find the surveys that are relevant to you.

Available parameters
  • sector - the sector/industry upon which the survey is based
  • ageGroup - the survey's target age group
  • country - the country upon which the survey was conducted
  • year - the year in which the survey was conducted

1.1.1 Single-parameter calls

Query format:

http://localhost:3000/api/surveys/{parameter}/{parameter-value}

Example request:

http://localhost:3000/api/surveys/sector/sportswear

1.1.2 Multi-parameter calls

Our multi-parameter calls allow you to filter your query by more than one parameter. If, for example, you want to fetch all gardening surveys from 2016, make a multi-parameter call.

Query format:

http://localhost:3000/api/surveys/search?{p1}={p1-value}&{p2}={p2-value}

Example request:

http://localhost:3000/api/surveys/search?sector=gardening&year=2016

Example response:
Notes

You can append any number of available parameters to your query, in any order.

For multi-word parameter values - where, for example, you want to view all surveys for the 'internet ads' sector - replace spaces with '%20'

All parameters and parameter values should be lower case.
